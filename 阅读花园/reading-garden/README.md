# Reading Garden

一个 Kindle 阅读伴侣网页。

**让读过的文字慢慢生长。**

### 页面

- **Garden** — 每日女性历史日历
- **Seasons** — 阅读计时 + 阅读日历 + 统计
- **Leaves** — 每日女性主义概念 + 精选文章

### 本地预览

```bash
cd reading-garden
python3 -m http.server 8765
```

浏览器打开 http://localhost:8765/

### 技术栈

纯 HTML + CSS + JavaScript (ES modules)。零依赖。零构建。
